# Package overview

## Important files
- `AGENTS.md`: agent mission and rules
- `IDENTITY.md`: role and persona
- `SOUL.md`: tone and safety boundaries
- `USER.md`: who the agent helps
- `TOOLS.md`: workflow and output format
- `WORKFLOW.md`: step-by-step triage flow
- `TRIAGE_RUBRIC.md`: priority rules and red flags
- `REPLY_TEMPLATE.md`: starter drafts
- `prompts/inbox-triage.md`: runtime prompt
- `openclaw.json.example`: install template

## Use
1. Copy the config example into the local OpenClaw config.
2. Point the agent at `workspace-inbox-triage-agent`.
3. Fill in credentials locally.
4. Keep human approval on outbound replies.
