# Customer Support Triage Workflow

## Flow
1) Inbound message arrives (email/Intercom/Zendesk/Slack).
2) Agent classifies category + priority.
3) Drafts a short response.
4) Escalates urgent issues for human review.

## Escalation Rules
- Security incidents
- Account access issues
- Payment failures / billing disputes
- Outage or critical bugs

## Notes
- Drafts only. Human approval required before sending.
